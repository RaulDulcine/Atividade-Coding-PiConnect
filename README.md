Atividade feita em sala

# **Desafio: Observatório de Projetos Integradores**

---

# **NÍVEL 1: Estrutura básica**

## **Objetivo**

Criar a base do sistema usando classes.

---

## **Enunciado**

Crie um sistema simples que permita cadastrar e listar Projetos Integradores.

---

## **Requisitos**

- Criar a classe `ProjetoIntegrador` com:
    - título
    - curso
    - período
- Criar a classe `Observatorio`:
    - lista de projetos
    - método para adicionar projetos
    - método para listar projetos

---

# **NÍVEL 2: Regras de negócio**

## **Objetivo**

Adicionar lógica ao sistema.

---

## **Enunciado**

Evolua o sistema para permitir:

- buscar projeto por título
- filtrar por curso
- validar título vazio

---

## **Requisitos**

- método `buscar_projeto(titulo)`
- método `filtrar_por_curso(curso)`
- validação no cadastro

---

# **NÍVEL 3: Organização e evolução**

## **Objetivo**

Preparar o sistema para uso real (interface/backend).

---

## **Enunciado**

Melhore o sistema adicionando:

- descrição do projeto
- encapsulamento
- método para alterar descrição
- método `to_dict()`
- listagem em formato de dicionário

---

## **Requisitos**

- usar `_descricao`
- criar `alterar_descricao()`
- criar `to_dict()`

---
